# Shell Environments

A shell is a command-line interpreter that provides a user interface for the operating system.

### Common Shells

- **Bash (`bash`)**: The Bourne Again Shell. This is the default shell for most modern Linux distributions (and what we use in CEG 2350).
    
- **Z Shell (`zsh`)**: A highly customizable shell, widely used and the default on macOS.
    
- **PowerShell**: The object-oriented command-line shell commonly used in Windows environments.
- **Fish**: simply goated!
