# `cd` (Change Directory)

Moves your terminal session into a different directory.

### Examples

- `cd DirA` : Move into a directory named DirA.
    
- `cd /home/puffy/"Dir B"` : Move into an absolute path with spaces (requires quotes).
    
- `cd ~` : Shortcut to return to your user's home directory.
    
- `cd ..` : Move up one directory level.
