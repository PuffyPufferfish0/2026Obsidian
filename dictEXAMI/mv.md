# `mv` (Move)

Moves files or directories. Because moving a file into the exact same directory with a new name effectively renames it, `mv` is also the standard Linux command for **renaming** files.

### Examples

- `mv test.txt /home/puffy/DirA` : Moves the file into `DirA`.
    
- `mv oldname.txt newname.txt` : Renames the file.
