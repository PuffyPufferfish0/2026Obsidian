
Navigating the filesystem via the terminal requires understanding your current location and how paths work.

### Key Concepts

- **Absolute Path**: The full path starting from the root directory (`/`). Example: `/home/puffy/DirA/test.txt`
    
- **Relative Path**: The path relative to your current directory. Example: `./test.txt` or `../DirA`
    
- **Home Directory (`~`)**: A shortcut to the current user's home path (e.g., `/home/puffy`).
    

### Essential Commands

- [[pwd]]: See where you are.
    
- [[cd]]: Change directories.
    
- [[ls]]: Look at the contents of the current directory.
    

### Lab 01 Examples

- `cd ~` : Return to user's home directory.
    
- `cd /home/puffy/"Dir B"` : Navigating into a directory with a space in the name (requires quotes).
