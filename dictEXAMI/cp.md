# `cp` (Copy)

Copies files or directories from a source to a destination.

### Examples

- `cp /home/puffy/DirA/test.txt /home/puffy/DirA/copy.txt` : Duplicates a file in the same directory.
    
- `cp test.txt "/home/puffy/Dir B/fodder.txt"` : Copies a file to a different directory with a new name.
