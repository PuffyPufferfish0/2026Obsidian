# `rm` (Remove)

Deletes files permanently (bypasses the trash can).

### Important Flags

- `-r` : Recursive. **Required** if you want to delete a directory and everything inside it.
    

### Examples

- `rm test.txt` : Deletes a file.
    
- `rm -r "/home/puffy/Dir B"` : Recursively deletes `Dir B` and all files inside it.
