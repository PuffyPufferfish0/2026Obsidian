# `vim` (Vi Improved)

A powerful, terminal-based text editor.

### Basic Usage Flow

1. Open file: `vim test.txt` (or `sudo vim` to edit protected files).
    
2. Enter **Insert Mode**: Press `i` or `a` to start typing text.
    
3. Enter **Command Mode**: Press `Escape`.
    
4. **Save and Quit**: In command mode, type `:wq` and press Enter.
    

### Lab Notes

- Unlike `notepad.exe` in Windows, `vim` opens directly inside the terminal window rather than launching a separate GUI application.
